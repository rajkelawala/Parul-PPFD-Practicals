How the API works

Click the button below and watch each step. You’ll see the request, the response, and the final list of users.

  1. You trigger the request Clicking the button runs JavaScript that calls the API.
  2. Browser sends GET /users The browser sends an HTTP GET request to the server.
  3. Server runs SQL and returns JSON Flask runs a query on the database and sends back JSON.
  4. We parse JSON and show users JavaScript receives the data and builds the cards below.
